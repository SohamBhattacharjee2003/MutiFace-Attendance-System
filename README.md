# PresenceAI 🚀  
### AI-Powered Face Recognition Attendance System

PresenceAI is a **smart, contactless attendance management system** that uses **deep learning–based face recognition** to automatically register students and mark attendance in real time with high accuracy.

This project integrates **Computer Vision, Machine Learning, and Full-Stack Web Development** to deliver a secure, scalable, and modern solution suitable for educational institutions and organizations.

---

## 📌 Key Features

- 📸 Camera-based student registration
- 🧠 Deep learning face recognition
- ⚡ Real-time attendance marking
- 🔐 Contactless & secure system
- 📊 Automatic attendance logging
- 🎨 Modern and responsive UI
- 🗂️ Student image management (add/update images)

---

## 🧠 Core Technologies Used

### 🔹 Frontend
- React.js  
- Vite  
- Tailwind CSS  
- Axios  
- HTML5 Camera API  

### 🔹 Backend
- Python (Flask)  
- Flask-CORS  
- MongoDB  
- Pillow (PIL)  

### 🔹 AI / Machine Learning
- **MTCNN** – Face Detection  
- **FaceNet** – Face Embedding Extraction  
- **Euclidean Distance / KNN / SVM** – Face Classification
